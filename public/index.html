My to-do server API

RESTful JSON API

Retrieve all to-dos

GET /to-dos

Create a to-do

POST /to-dos

Retrieve a specific to-do

GET /to-dos/{id}

Update a specific to-do

PUT /to-dos/{id}

Attaching a JSON body with the possible below fields:

  • title (string)
  • message (string)

Delete a specific to-do

DELETE /to-dos/{id}
